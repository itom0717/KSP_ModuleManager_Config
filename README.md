# KSP_ModuleManager_Config
====

KSP ModuleManager.DLL用のConfigファイルです。  

・謎のバッテリー
　通常の100倍の容量のバッテリー

・謎のソーラーパネル
　通常の100倍の発電量のソーラーパネル

・謎の原子力電池
　通常の100倍の発電量の原子力電池

・謎の燃料タンク
　電気から燃料が生成する

・謎のリソース生成装置
　電気から燃料が生成する
